Input Voter

Data must be strictly formatted as follows:

  • Each voter’s record must be on a line of its own.
  • Provide exactly eight pieces of data for each voter, with a semi-colon between each piece. Those pieces are:
    1. First name
    2. Last name
    3. Email address
    4. Username (When available.)
    5. Password (When available.)
  • when a piece of data is unavailable, provide a single blank space in its place. For example, see in the example at the bottom, there’s no email address for Shelly Harrison, but rather, a blank space where the email address would have appeared.
